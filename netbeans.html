Netbeans IDE

Aplikasi desktop berbasis Java yang dirancang untuk membantu pengelolaan data program studi melalui antarmuka grafis (GUI). Sistem menyediakan fitur CRUD, navigasi data, serta pencetakan laporan untuk mendukung pengelolaan data akademik secara efektif. Pengembangan dilakukan menggunakan Java Swing dengan menerapkan konsep Object-Oriented Programming (OOP).
